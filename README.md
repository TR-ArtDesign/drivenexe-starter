# Project Base

Structured project starter with:

* roadmap system
* validation layer
* testing
* CI
* agent integration

## Runtime

This project uses **bun** as the primary runtime.

Commands:

```bash
bun install
bun run validate
bun run test:e2e
```

npm is not required.

## Tipos de projeto suportados

* Web → testes com Playwright
* Mobile → testes com Maestro
* Multiplataforma → ambos separados
