# Landing Page Module

Este é um módulo isolado para a landing page do projeto.

## Estrutura

- `src/app/`: Lógica principal da aplicação.
- `src/components/`: Componentes React reutilizáveis.
- `src/styles/`: Configurações de estilo (Tailwind CSS).

## Como rodar

```bash
cd apps/landing
bun install
bun run dev
```

## Características

- React 18+
- Vite (Fast Build Tool)
- Tailwind CSS (Styling)
- Isolamento total da fundação do projeto.
